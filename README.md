# project_sprint_7
Proyecto aplicacion web

link a la aplicación: https://project-sprint-7-syap.onrender.com

Descripción de la app:
La aplicación web para este proyecto ha sido realizada con un dataset que contiene información sobre vehículos usados en venta, incluye precio, año de modelo, condición, cilindros, tipo de combustible, kilometraje, tipo de transmisión, tipo de vehículo, color, tracción en las cuatro ruedas, fecha de publicación y días en el mercado. 

Analizando las dos columnas que nos interesan son: kilometraje (odometer) y precio (price). 

El histograma muestra que la mayoría de vehículos tienen un uso moderado en cuanto a kilometraje, alrededor de las 100,000 millas. 

El gráfico de dispersión utilizando el kilometraje y precio, por su parte muestra una tendencia descendente. Es decir, en general, los vehículos con mayor kilometraje tienden a tener precios más bajos. Esta es una tendencia común en el mercado de vehículos usados.
